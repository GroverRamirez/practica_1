# Programa Académico: Desarrollo de Software con Laravel

Bienvenido al directorio documental de la práctica. Este proyecto está diseñado para guiar a los estudiantes en el desarrollo de una plataforma corporativa "Desde Cero", escalando desde las bases tecnológicas hasta la implementación de una interfaz de usuario Premium y patrones de pruebas.

## Estructura del Curso (Laboratorios)

A continuación, sigue las guías en orden estricto para ensamblar el sistema completo:

1. **[01_Modulo_Autenticacion_Breeze.md](./01_Modulo_Autenticacion_Breeze.md)**  
   *Instalación del sistema, conexión a base de datos y levantamiento del Login/Registro base.*
2. **[02_Modulo_Categorias_CRUD.md](./02_Modulo_Categorias_CRUD.md)**  
   *Desarrollo del primer módulo para entender Controladores, Vistas Blade y Modelos (ORM).*
3. **[03_Modulo_Productos_ForenKeys.md](./03_Modulo_Productos_ForenKeys.md)**  
   *Manejo de relaciones (Una categoría -> Muchos productos) y Migraciones complejas.*
4. **[04_Modulo_Busqueda_Paginacion.md](./04_Modulo_Busqueda_Paginacion.md)**  
   *Optimizaciones de listados de gran volumen de datos, inyecciones seguras y query strings.*
5. **[05_Modulo_Reportes_PDF.md](./05_Modulo_Reportes_PDF.md)**  
   *Integración con el motor DomPDF para exportación contable y ofimática.*
6. **[06_UI_Premium_Seeders.md](./06_UI_Premium_Seeders.md) (🏆 Avanzado)**  
   *Nivelación a Senior UI UX: Implementación de componentes Glassmorphism, tipografía Inter, y motores de llenado automatizado (`DatabaseSeeders` y `Factories`).*

---
> **Nota para los Docentes:** Asegúrese de que los alumnos superen el Laboratorio 05 antes de intentar implementar los patrones de diseño del Laboratorio 06, ya que este último requiere un conocimiento sólido previo del motor ORM de Laravel.
